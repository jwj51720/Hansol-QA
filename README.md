# Hansol-QA
도배 하자 질의 응답 처리 : 한솔데코 시즌2 AI 경진대회
[Dacon](https://dacon.io/competitions/official/236216/overview/description)

- `feat`: 새로운 기능
- `exp`: 실험
- `refactor`: 코드 리팩토링
- `docs`: 문서 작업 (README.md, docstring, type hint 등)
- `style`: 코드 포맷팅 관련 작업(pep8, flake8 등)
- `fix`: 버그, 오류 수정
- `chore`: 잡다한 일들. 코드 변경 없음.
